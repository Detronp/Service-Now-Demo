Demo Steps
1. Instance Ready

The ServiceNow account and instance are already set up.

Note: Instance creation can take time, so this step is intentionally skipped to keep the demo efficient and focused on workflow execution.

2. Dashboard Overview

Log into the ServiceNow instance

Briefly review the dashboard

Highlight key widgets or modules to show familiarity with the interface

This provides context before transitioning into ticket creation.

3. Navigate to Incidents

Click All in the application navigator

Search for Incident

Select Incident from the menu

This opens the Incident list view.

4. Create a New Incident Ticket

Click New to create a ticket

Fill out the required fields, including:

Short Description – Summary of the issue

Description – Detailed explanation of why the customer is calling

This step demonstrates proper documentation and communication skills.

5. Submit the Ticket

Review the entered information for accuracy

Click Submit

The incident is now logged in the system and ready for assignment and resolution.

Outcome

Incident successfully created

Ticket appears in the Incident list

Demonstrates ability to navigate ServiceNow and execute a core ITSM function# Service-Now-Demo
